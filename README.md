## Design Philosophy

This isn't another cookie-cutter barbershop website. The design focuses on:

- **Craftsmanship over commodity**: Positioning Ricky as a master craftsman, not just a barber
- **Single chair luxury**: Highlighting the intimate, personalized experience
- **Traditional meets modern**: Classic barbering with contemporary web design
- **Visual storytelling**: Let the work and reviews speak for quality

## Technical Details

- **Built with**: Vanilla HTML, CSS, JavaScript
- **No dependencies**: Self-contained, fast-loading
- **Fonts**: Playfair Display (serif) + Source Sans Pro (sans-serif)
- **Performance**: Optimized for mobile, lightweight assets
- **Accessibility**: Semantic HTML, proper contrast ratios
